# Anotacoes_wtime


- Instalação node
  
- node -version

- Intalação do git
    
- git --version

- install --global yarn

- yarn --version

- Adicionar pacote - yarn add axios

- Dev dependency - yarn add -D jest

- Rodar script localmente	- yarn dev

- Para publicar o projeto → yarn build depois yarn start

- Para checar erros de código → yarn lint
